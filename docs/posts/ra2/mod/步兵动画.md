## 基本状态

|     名称      | 单一方向 | 说明                                                              |
| :-----------: | :------: | :---------------------------------------------------------------- |
|     Ready     |    ❌    | 就绪                                                              |
|     Guard     |    ❌    | 警戒                                                              |
|     Walk      |    ❌    | 行走[常规移动]                                                    |
|    FireUp     |    ❌    | 指定当步兵单位站着向敌方单位开火的帧序列。                        |
| SecondaryFire |    ❌    | （仅用于尤里的复仇）和FireUp=相同，不同的是这里使用第二武器开火。 |
|     Idle1     |    ✔    | 待机动画1                                                         |
|     Idle2     |    ✔    | 待机动画2                                                         |
|     Die1      |    ✔    | 被枪弹击中时的死亡动画                                            |
|     Die2      |    ✔    | 爆炸时的死亡动画                                                  |
|     Die3      |    ✔    | 爆炸时的死亡动画（备用）                                          |
|     Die4      |    ✔    | 震荡爆炸时的死亡动画                                              |
|     Die5      |    ✔    | 火灾中的死亡动画                                                  |
|     Cheer     |    ✔    | 指定步兵单位欢呼的帧序列。                                        |
|   Paradrop    |    ✔    | 指定当步兵单位被空投到战场上的帧。该帧不要有影子。                |
|     Panic     |    ❌    | 惊恐动画                                                          |
